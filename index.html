Learning embeddings with message passing

Message passing

Message passing allows nodes to communicate (update each other) via edges.

  1. Data on adjacent nodes, plus edge data, concat’d with each other
  2. Run through trainable NN layer \(\to\) new edge embedding
  3. All edges incident on node concat’d and run through another NN layer
  4. Permutation invariant aggregate, eg. sum, mean, etc., forms message
  5. Message concat’d with node data, then run through NN for node embedding

Promising results

  • Interaction networks learn relations between particles 1

    • Boosted W-boson clustering [3]
  • Attention based GNNs improve background rejection [4]
  • Initial work to reproduce results of [3], but research now pivoting towards attention based techniques, and variational graph autoencoders (VGAEs) [5]

VGAEs promising qualities

  • Can extract knowledge from rich simulation-based truth
    • If using for compression, input data can be arbitrarily rich
    • If disentangled, suggest known or new high level observables
  • Versatile: can be used for both edge and node level prediction
  • Generative: sample from Normal dist’bn can be used to generate new events
    • Possibility to interpolate between kinematic properties of collisions
